## 小白使用说明

如果你是第一次使用 GitHub，可以先记住这几个词：

- **Repository / 仓库**：放项目文件的地方
- **Branch / 分支**：临时修改内容的地方
- **Commit / 提交**：保存一次修改
- **Pull Request / PR**：把修改提交给主分支前的确认流程
